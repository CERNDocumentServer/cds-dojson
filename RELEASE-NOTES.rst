==================
CDS-DoJSON v0.2.0
==================

DoJSON v0.2.0 was released on February 29, 2016.

About
-----

CDS DoJSON is an extension of DoJSON package with the customization for CDS.

Installation
------------

   $ pip install cds-dojson==0.2.0
