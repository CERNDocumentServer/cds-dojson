==================
CDS-DoJSON v0.3.3
==================

DoJSON v0.3.3 was released on 3 Octomber, 2016.

About
-----

CDS DoJSON is an extension of DoJSON package with the customization for CDS.

New features
------------

- Video & Project schemas
- CLI for resolving schemas 

Installation
------------

   $ pip install cds-dojson==0.3.3
